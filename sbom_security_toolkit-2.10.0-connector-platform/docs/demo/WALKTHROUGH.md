# Demo Walkthrough

1. Run `make demo-product`.
2. Start the Workbench with `make ui-server`.
3. Open `http://127.0.0.1:8080`.
4. Visit Reports to browse generated demo artifacts.
5. Visit Findings to review remediation workflows.
6. Visit AI Reports to generate an executive or engineering report.
7. Visit Integrations to create dry-run Jira/DefectDojo/GitHub artifacts.

The demo creates sample files under `reports/demo-product/`.
