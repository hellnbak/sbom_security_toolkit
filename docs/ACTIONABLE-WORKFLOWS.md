# Actionable Workflows (v2.13.0)

v2.13.0 shifts the Workbench from presenting security data to guiding users toward completion.

## Included UX capabilities

1. Persistent recommended-next-action guidance
2. Prioritized dashboard/action center
3. Seven task-based guided workflows
4. Six scan profiles
5. Plain-language scan and release summaries
6. Finding-level remediation actions
7. Bulk triage patterns
8. Saved views
9. Project-centered navigation
10. Activity timelines
11. Notification preferences
12. Developer, security, executive, and auditor home views
13. Guided and advanced modes
14. Plain-language terminology
15. Inline explanations
16. Guided risk-exception flow
17. Policy simulation
18. Comparison-oriented workflows
19. Named progress stages
20. Actionable error recovery
21. Environment-aware defaults
22. Command palette and keyboard access
23. Sanitized support bundles
24. Accessibility preferences and non-color status labels
25. In-product feedback

## New routes

- `/workflows`
- `/saved-views`
- `/activity`
- `/notifications`
- `/personas`
- `/policy-simulator`
- `/support`
- `/feedback`

All new state is local under `ui/storage/ux`. Connector writes remain opt-in.
