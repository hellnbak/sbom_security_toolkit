# GUI Feature Coverage

| Capability | GUI location | Default behavior |
|---|---|---|
| SBOM upload and analysis | Upload / Run Analysis | Local job |
| Repository intake | Repository Intake | Local or opt-in remote |
| Project history and trends | Projects | Read-only view |
| Findings and remediation | Findings / Security Controls | Plan-only |
| Release assurance | Release Decisions / Security Controls | Local evaluation |
| Risk exceptions | Exceptions / Settings | Local governance file |
| VEX | Security Controls / Integrations | Draft export |
| Provenance | Security Controls | Local digest verification |
| Evidence bundles | Evidence / Security Controls / Job page | Local generation |
| Organization context | Security Controls / Projects | Template generation |
| Connectors | Connectors | Read-only and dry-run by default |
| Reports and AI reports | Reports / AI Reports | Local generation |
| Policies and settings | Policies & Settings | Local YAML management |
| Fuzzing and AI fuzzing | Fuzzing Lab | Time-bounded local jobs |
| Scanner compatibility | Scanner Status | Read-only probe |
| Administration and RBAC scaffolding | Administration | Local configuration |
